# Django-Blog
基于Django的个人博客/网站系统

如题，这个是我初学Django框架的作品。它是一个包含多种功能的个人网站/blog，甚至还集成了微信公众号机器人的后端服务。它现在被运行于一台阿里云ECS轻量级云服务器上，你可以通过http://www.saher666.cn/ 访问它。
